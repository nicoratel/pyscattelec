This package contains files for the simulation of kinematic and dynamic scattering from structural models (xyz files).

Tools are developped to build a powder of nanoparticles from the descrption of a single particle.

**Kinematic scattering** is computed with Debye scattering equation.

**Dynamic scattering** is computed using multi slice approach available in abtem package.

Additional functions allow the extraction of pari distribution function from scattering data obtained with kinematic and dynamic frameworks.
